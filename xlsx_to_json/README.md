# 🚂 XLSX → JSON Конвертер для КБШ

Инструменты для конвертации таблицы скоростей из Excel в JSON и группировки данных.

## 📁 Файлы

| Файл | Назначение |
|------|------------|
| `excel-to-json-kbsh-v2.py` | Конвертация XLSX → JSON (плоская структура) |
| `group-by-sections-v2.py` | Группировка JSON по участкам (иерархия) |
| `excel-converter.html` | Браузерный конвертер XLSX → JSON |

## 🔄 Рабочий процесс

### Шаг 1: Конвертация XLSX в JSON
```bash
python excel-to-json-kbsh-v2.py "data/КБШ Н-122 от 16.09.2025 (2) обновлен.xlsx"
```

**Результат:** Плоский JSON файл с данными

### Шаг 2: Группировка по участкам
```bash
python group-by-sections-v2.py
```

**Результат:** `data/участки-иерархия-v2.json` с иерархической структурой:
```
Участок
├── Станция
│   ├── Пути
│   └── Съезды
└── Перегон
    └── Пути
```

### Шаг 3: Очистка от \n (при необходимости)
```bash
python clean-json-newlines.py
```

**Результат:** JSON без символов новой строки в текстовых полях

## 📊 Структура данных

### Исходные данные (Excel)
- Таблица с колонками: № п/п, Путь, Наименование, Протяженность, Скорости, Примечание
- Многострочная шапка
- Смешанные данные (станции, перегоны, пути, съезды)

### После конвертации (JSON)
```json
{
  "metadata": {
    "всего_участков": 56,
    "всего_объектов": 876,
    "всего_станций": 452,
    "всего_перегонов": 424,
    "всего_путей": 1755,
    "всего_съездов": 1533
  },
  "участки": [
    {
      "название": "Участок 1638 км- Шугуровка",
      "объекты": [
        {
          "название": "Пп 1638 км",
          "тип": "станция",
          "пути": [...],
          "съезды": [...]
        }
      ]
    }
  ]
}
```

## 🎯 Особенности

- ✅ Автоматическое определение заголовков таблицы
- ✅ Поддержка многострочной шапки
- ✅ Сохранение кириллицы (UTF-8)
- ✅ Обработка дополнительных сегментов путей
- ✅ Группировка по станциям и перегонам
- ✅ Сохранение порядка следования объектов

## 📝 Требования

- Python 3.6+
- Никаких внешних зависимостей (используются встроенные `zipfile`, `xml.etree`, `json`)
