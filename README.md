# 🚂 Калькулятор норм расхода электроэнергии РЖД

[![CI/CD Pipeline](https://github.com/zimixin/zimixin.github.io/actions/workflows/ci-cd.yml/badge.svg)](https://github.com/zimixin/zimixin.github.io/actions/workflows/ci-cd.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PWA](https://img.shields.io/badge/PWA-ready-blue)](https://pwa-directory.appspot.com/)

Веб-калькулятор для расчёта норм расхода электроэнергии и топлива в грузовом движении на тягу поездов. PWA-приложение с динамической загрузкой маршрутов и поддержкой различных типов локомотивов.

🌐 **Демо**: [zimixin.github.io](https://zimixin.github.io)

## 🧮 Формула расчёта

```
Норма расхода (кВт⋅ч) = (Вес поезда × Коэффициент × Расстояние) / 10000 / 100
```

| Параметр | Описание |
|----------|----------|
| **Вес поезда** | Масса состава в тоннах |
| **Коэффициент** | Зависит от типа локомотива и нагрузки на ось (индивидуальный для маршрута или стандартный) |
| **Расстояние** | Длина маршрута в километрах |


## 🏗️ Архитектура

Подробная документация архитектуры доступна в [`ARCHITECTURE.md`](./ARCHITECTURE.md).

### Ключевые компоненты

| Компонент | Файл | Ответственность |
|-----------|------|-----------------|
| **EnergyCalculator** | `calculator.js` | Расчёты, валидация, история, экспорт |
| **ThemeManager** | `themes.js` | Переключение тем, сохранение настроек |
| **RouteLoader** | `data.js` | Загрузка и парсинг маршрутов, коэффициенты |
| **PWAManager** | `pwa.js` | Офлайн режим, кэширование, уведомления |

### C4-модель

```
┌──────────────┐      HTTPS      ┌──────────────────┐
│   Пользователь│────────────────▶│  Калькулятор РЖД │
│  (Браузер)   │◀───────────────│                  │
└──────────────┘                 └──────────────────┘
                                          │
                    ┌─────────────────────┼─────────────────────┐
                    ▼                     ▼                     ▼
          ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
          │  LocalStorage   │  │  Service Worker │  │  Файлы /data    │
          │  (История,      │  │  (Кэш, офлайн)  │  │  (Маршруты)     │
          │   Настройки)    │  │                 │  │                 │
          └─────────────────┘  └─────────────────┘  └─────────────────┘
```
